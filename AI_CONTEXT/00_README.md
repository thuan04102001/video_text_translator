# AI Context Index

Thu muc nay gom toan bo tai lieu ban giao cho Codex/AI agent khac. Khi doi tai khoan ChatGPT/Codex, hay yeu cau agent moi doc cac file theo thu tu ben duoi truoc khi tiep tuc code.

## Thu tu doc nhanh

1. `01_PROJECT_CONTEXT.md` - muc tieu, stack, module, trang thai xu ly video.
2. `02_SYSTEM_OVERVIEW.md` - luong app va pipeline tong quan.
3. `03_ARCHITECTURE.md` - cau truc backend/frontend/runtime.
4. `04_WORKFLOW_RULES.md` - rule bat buoc khi sua du an.
5. `05_DECISIONS.md` - cac quyet dinh san pham/ky thuat quan trong.
6. `06_CHANGELOG.md` - snapshot cac thay doi da lam.
7. `07_KNOWN_BUGS.md` - rui ro va diem can canh giac.
8. `08_CURRENT_TASK.md` - trang thai cong viec gan nhat.
9. `09_NEXT_TASK.md` - viec tiep theo du kien.
10. `10_TODO.md` - backlog ngan/trung han.

## Nguyen tac root

Root du an chi nen giu file van hanh can thao tac nhanh nhu `README.md`, `setup.bat`, `start.bat`, `clear_cache.bat`. Tai lieu dai han de trong thu muc nay de tranh roi mat.
